# 注意：CuPy需要根據CUDA版本選擇安裝
# 請在安裝基本依賴後，根據您的CUDA版本選擇一個：
#
# CUDA 11.x 用戶請執行：
# pip install cupy-cuda11x>=10.0.0
#
# CUDA 12.x 用戶請執行：
# pip install cupy-cuda12x>=11.0.0
#
# ROCm (AMD GPU) 用戶請執行：
# pip install cupy-rocm-5-0>=10.0.0
#
# 無GPU或僅CPU模式用戶：
# 無需安裝CuPy，系統會自動回退到NumPy
